# ⚡ CyberTemp X – Premium Temperature Suite

CyberTemp X is a futuristic desktop application built using Python and CustomTkinter. It provides an advanced temperature conversion experience with a cyberpunk-inspired user interface, real-time calculations, voice feedback, animated visualizations, PDF report generation, historical analytics, and an integrated scientific calculator.

---

## 🌟 Features

### 🌡️ Temperature Converter
- Real-time temperature conversion
- Supports Celsius, Fahrenheit, and Kelvin scales
- Adjustable decimal precision
- Temperature preset shortcuts
- Swap source and destination units
- Copy converted values to clipboard
- Favorites management system

### 🎨 Animated Thermometer
- Dynamic vector-based thermometer
- Color changes based on temperature range
- Smooth fluid animation effects

### 🔊 Voice Assistance
- Announces converted temperature values using text-to-speech
- Powered by **pyttsx3**

### 📈 Data Visualization
- Embedded Matplotlib trend graph
- Tracks conversion history over time
- Interactive history filtering

### 📝 History Management
- Stores previous conversions
- Search and filter logs
- Export history as TXT files
- Export history as CSV files
- Clear history database

### 📂 PDF Report Generator
- Generates professional PDF reports
- Includes timestamps and conversion summaries
- Built using ReportLab

### 🧮 Scientific Calculator
Supports:

- Addition
- Subtraction
- Multiplication
- Division
- Powers (^)
- Square Root
- Sine
- Cosine
- Tangent
- Logarithm
- Parentheses operations

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/CyberTemp-X.git

cd CyberTemp-X
```

Install required packages

```bash
pip install customtkinter
pip install matplotlib
pip install pyttsx3
pip install reportlab
```

Or install everything together

```bash
pip install customtkinter matplotlib pyttsx3 reportlab
```

---

## ▶️ Running the Application

```bash
python main.py
```
